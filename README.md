> **600,000+ people** are released from US prisons annually. Within 3 years, **2 in 3 are rearrested**. The first 30 days are critical — housing, employment, and benefits navigation can break the cycle.

## The Solution

3 AI agents with dignity-first design. Named agents (Alex, Jordan, River) because people deserve to interact with something that feels human. GPL-3.0.

## Agents

**HomeBase (Alex)** — Housing placement within 30 days. Transitional housing, halfway houses, sober living.
**PathFinder (Jordan)** — Employment within 60 days. Skills translation, ban-the-box employers, apprenticeships.
**LifeLines (River)** — Benefits enrollment within 14 days. SNAP, Medicaid, ID restoration.

## Safety

MANDATORY crisis handoffs. Trauma-informed design. Dignity-first language. No PII retained beyond session.

## Data Sources

DOJ Second Chance Act resources, state reentry councils, Legal Services Corporation
